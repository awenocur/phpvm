
phpvm - A php version management system

-- Commands

  phpvm reload

    Reloads the script's environment

  phpvm install {version_number}

    Installs a php version

  phpvm use {version_number}

    Sets a php bin directory in the path to use for the current session
    Use should only activate one bin directoy at a time

  phpvm list

    See a list of php versions installed

  phpvm cleanup

    Forcefully deletes the ~/.phpvm/tmp directory which consists of downloaded
    files and source code
